# 🎧 Spotify Listener Intelligence System

This project uses unsupervised machine learning to segment music streaming users based on listening behavior and engagement patterns.

---

## 📊 Project Overview

A synthetic dataset of 20,000 users was generated to simulate realistic Spotify listening behavior.

The system identifies distinct listener segments using clustering techniques.

---

## ⚙️ Key Steps

- Dataset generation (synthetic user behavior)
- Feature engineering (genre distribution + diversity)
- Feature scaling (StandardScaler)
- Clustering (K-Means)
- Model evaluation (Elbow Method + Silhouette Score)
- PCA visualization

---

## 🎯 Final Listener Segments

1. **Power Explorers**
   - High engagement
   - High diversity
   - Low skip rate

2. **Genre Loyalists**
   - Moderate engagement
   - Low diversity
   - Consistent listening

3. **Low Engagement Users**
   - Low activity
   - High skip rate
   - High churn risk
